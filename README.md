## ja
[LuaJIT](https://github.com/LuaJIT/LuaJIT)のv2.1ブランチの非公式ビルドです。

AviUtlでの利用を前提としてx86のみをビルドしています。

## For AviUtl users
Releasesから`LuaJIT_2.1_Win_x86.zip`をダウンロードし、中身のlua51.dllを付属のものと置き換えてください。
